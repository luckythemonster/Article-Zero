# Dimensions of UI elements made with the v0 MCP

The dimensions of the Door Panel (`DoorKeypad.tsx`) and Wall Terminal (`WallTerminal.tsx`) UI elements based on the CSS generated by the v0 MCP (`src/index.css`) are as follows:

## Overall Panel Dimensions (`.wall-terminal`)
- **Width:** `min(420px, 92vw)`
- **Height:** Dynamic (auto-sizing based on content). The overall container (`.overlay-panel`) limits the maximum height to `92%` of the viewport (or `calc(100% - 252px)` on touch devices).

## Sub-Components
- **Keypad Buttons (`.wall-terminal__key`):**
  - Height: `3rem` (48px)
  - Width: Auto-sizes within a 3-column CSS Grid (`grid-template-columns: repeat(3, 1fr)`)
- **Code Display Characters (`.wall-terminal__display-char`):**
  - Height: `2.4rem`
  - Width: Dynamically sized evenly (`grid-auto-columns: 1fr`)
- **Climate Control Arrows (`.wall-terminal__arrow`):**
  - Width: `48px`
  - Height: `48px`
